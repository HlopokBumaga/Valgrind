Внимание! Программа создана в рамках моего выпускного проекта. Использовать её в реальных условиях для шифрования действительно важной информации запрещено.

### **Шифрование текстовой информации**

Алгоритм шифрования:
1. В главном меню выбрать режим шифрования текста.

2. Выбрать действие и ввести данные для шифрования.

```
Действия:

Зашифровать - зашифровывает данные, используя ключ, введённый пользователем.

Расшифровать - расшифровывает данные, используя ключ, введённый пользователем. Инвертирует ключ (умножает на -1) автоматически (для шифра Цезаря и шифра Виженера).

Для XOR шифрования разницы между действиями нет.
```

3. Выбрать метод шифрования из выпадающего списка.

4. Ввести ключ два раза.

5. Подтвердить шифрование и получить результат.

### **Возможные ошибки**

- ```invalid method: the encryption method is not selected``` - не выбран метод шифрования.
- ```Confirmation error: keys do not match``` - ключи не совпадают.
- ```invalid data: there is nothing to encrypt``` - не введены данные для шифрования.
- ```invalid key: the key must be ...``` - некорректный ключ.
