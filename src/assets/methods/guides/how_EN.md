### **Encryption of text information**

Encryption algorithm:
1. In the main menu, select the text encryption mode.

2. Select an action and enter the data for encryption.

```
Actions:

Encrypt - Encrypts data using the key entered by the user.

Decrypt - decrypts data using the user-entered key. Inverts the key (multiplies by -1) automatically (for the Caesar cipher and Vigenère cipher).

For XOR encryption, there is no difference between the actions.
```

3. Select the encryption method from the drop-down list.

4. Enter the key twice.

5. Confirm encryption and get the result.
