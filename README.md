# Valgrind

### About

Valgrind is a program for encrypting information for educational purposes. It allows you to encrypt and decrypt various types of information.

### Installation

</div>

1. Clone the repository:
```
git clone https://github.com/HlopokBumaga/Valgrind.git
cd Valgrind
```
2. Create a viral environment.
3. Install the dependencies:
```
pip install flet["all"]
```

<div align="center">

### Usage

</div>

Launch main.py:
```
cd src
python main.py
```
